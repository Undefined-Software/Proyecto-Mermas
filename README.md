# Proyecto-Mermas

## FRONTEND

### Inicializar el proyecto
Entrar en una terminal dentro de la carpeta _Mermas-Frontend_ y usar el comando:
```
npm install
```
Esto para agregar la carpeta node_modules al proyecto de Angular
